# 🌦️ Weather App

A simple and responsive Weather App that shows the current weather and forecast using live weather data from a public API (like OpenWeatherMap or WeatherAPI).

## 📸 Preview

![Weather App Screenshot](screenshot.png)

## 🚀 Features

- 🔍 Search weather by city
- 📍 Get weather for current location
- 🌡️ View temperature, feels like, humidity, wind speed, and conditions
- 🌓 Dark and light theme toggle
- 📱 Responsive design for mobile and desktop

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React
- **API:** OpenWeatherMap API
- **Icons:** Font Awesome / Weather Icons

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/yourusername/weather-app.git
cd weather-app

# Install dependencies (if using a frontend framework)
npm install
```

# Run Application Using:(for React/Vite/Node projects)

    In Bash/Terminal:
        npm run dev
    Or
        just open index.html in a browser if it's a plain HTML/JS project.

# 🔑 API Key Setup:

    Get a free API key from [OpenWeatherMap](https://openweathermap.org/api).

    Create a `.env` file in the root directory and add:

    ```
    VITE_API_KEY=your_api_key_here
    ```

    Replace `your_api_key_here` with your actual API key.

        // Example
        const API_KEY = "your_api_key_here";

If using .env files, create one and add:

    VITE_API_KEY=your_api_key_here
